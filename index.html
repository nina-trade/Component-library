How to refresh

The theme moves; this site doesn't auto-sync. To pull in the latest components and tokens, open this folder in Claude and paste: 

Refresh the Trade component library — pull current sections, snippets, design tokens, regenerate all pages, and show me a changelog.

Re-generation runs through Claude (because Shopify theme data is only reachable via the MCP tools). After a regenerate, run git push and GitHub Pages will update.
